Privacy Policy

Last Updated: January 31, 2026

Core Commitment: Offline Password Manager is a completely offline password management tool. We do not collect, store, or transmit any of your personal data.

1. Data Collection

We do not collect any user data, including but not limited to:

  • Personally identifiable information (name, email, phone, etc.)
  • Passwords or other confidential information
  • Device information
  • Usage statistics
  • Crash logs
  • Location information

2. Data Storage

All password data is encrypted using the AES-256 encryption algorithm and stored exclusively locally on your device. Specific storage locations:

  • Encrypted data file: Stored in the app sandbox's Documents directory
  • Master password (if biometric authentication is enabled): Stored in the iOS system Keychain, protected at the system level

This data:

  • Will not be uploaded to any server
  • Will not be transmitted over the network
  • Will not be shared with any third party
  • Is accessible only by you

3. Data Security

We employ the following security measures to protect your data:

  • AES-256 Encryption: Military-grade encryption standard
  • Master Password Protection: Uses the PBKDF2 key derivation algorithm
  • Biometric Authentication: Supports Face ID/Touch ID, with the password stored in the system Keychain
  • Local Storage: Data never leaves your device

4. Network Connectivity

This application operates entirely offline and requires no network permissions. It will not:

  • Connect to the internet
  • Access any servers
  • Send any data
  • Receive any data

5. Third-Party Services

This application does not use any third-party services, including:

  • Analytics services (e.g., Google Analytics)
  • Advertising services
  • Crash reporting services
  • Cloud storage services

6. Permissions

This application only requests the following necessary permissions:

  • Face ID/Touch ID (Optional): Used to quickly unlock the app, and requires your explicit enablement

We do not request the following permissions:

  • Network access
  • Location information
  • Camera / Photo Library
  • Contacts
  • Microphone

7. Data Export and Deletion

  • Export: You can export your encrypted data file at any time via the "Export Data" feature within the app
  • Deletion: Uninstalling the app will permanently delete all local data, and it cannot be recovered

8. Children's Privacy

This application is not directed at children under the age of 13, and we do not knowingly collect personal information from children.

9. Privacy Policy Changes

If we update our privacy policy, we will post the new version on this page and update the "Last Updated" date. Major changes will be notified to users within the app.
