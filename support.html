常见问题

如何重置主密码？

重要提示：由于安全设计，主密码无法重置或找回。

如果忘记主密码，唯一的解决方案是：

  1. 删除应用
  2. 重新安装
  3. 设置新的主密码
⚠️ 警告：删除应用将永久丢失所有密码数据，且无法恢复！请务必牢记主密码，或使用导出功能定期备份。

如何备份数据？

使用应用内的"导出数据"功能：

  1. 打开应用，进入"设置"
  2. 点击"导出/导入数据"
  3. 点击"导出数据"
  4. 输入导出密码（可以与主密码不同）
  5. 保存导出的文件到安全位置（如 iCloud Drive）

如何导入数据？

  1. 打开应用，进入"设置"
  2. 点击"导出/导入数据"
  3. 点击"导入数据"
  4. 选择之前导出的文件
  5. 输入导出时设置的密码

注意：导入会覆盖当前所有数据，请谨慎操作。

Face ID/Touch ID 无法使用？

请检查：

  • 设备是否支持 Face ID/Touch ID
  • 系统设置中是否已启用生物识别
  • 是否已在应用设置中启用生物识别
  • 是否已录入面容/指纹数据

如果仍无法使用，可以使用主密码登录。

数据存储在哪里？

所有数据仅存储在您的设备本地：

  • 加密数据文件：应用沙盒 Documents 目录
  • 主密码（如启用生物识别）：iOS 系统 Keychain

数据不会上传到云端或任何服务器。

如何更改主密码？

  1. 打开应用，进入"设置"
  2. 点击"更改主密码"
  3. 输入当前主密码
  4. 输入新密码并确认

更改后，如果已启用生物识别，会自动更新 Keychain 中的密码。

应用安全吗？

是的，我们采用多重安全措施：

  • AES-256 加密：军事级别加密标准
  • PBKDF2 密钥派生：防止暴力破解
  • 完全离线：数据永不离开设备
  • 系统 Keychain：生物识别密码受系统保护
  • 无网络权限：无法连接互联网

支持 iCloud 同步吗？

不支持。为了最大化安全性，本应用完全离线运行，不支持任何云同步功能。

如需在多设备间同步，请使用"导出/导入"功能手动传输数据。

故障排除

应用崩溃或无响应

  1. 强制退出应用
  2. 重启设备
  3. 重新打开应用

如果问题持续，请联系技术支持。

无法保存新密码

请确保：

  • 标题和密码字段不为空
  • 设备存储空间充足
  • 应用有写入权限
