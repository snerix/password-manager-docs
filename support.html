Frequently Asked Questions

How do I reset my master password?

Important Note: Due to security design, the master password cannot be reset or retrieved.

If you forget your master password, the only solutions are:

  1. Delete the app
  2. Reinstall the app
  3. Set a new master password
⚠️ Warning: Deleting the app will result in the permanent loss of all password data, which cannot be recovered! Please ensure you memorize the master password or use the export feature to perform regular backups.

How do I backup my data?

Use the "Export Data" feature within the app:

  1. Open the app and go to "Settings"
  2. Tap "Export/Import Data"
  3. Tap "Export Data"
  4. Enter an export password (can be different from the master password)
  5. Save the exported file to a secure location (e.g., iCloud Drive)

How do I import data?

  1. Open the app and go to "Settings"
  2. Tap "Export/Import Data"
  3. Tap "Import Data"
  4. Select the previously exported file
  5. Enter the password set during export

Warning: Importing will overwrite all current data, please proceed with caution.

Face ID/Touch ID is not working?

Please check:

  • Whether the device supports Face ID/Touch ID
  • Whether biometrics are enabled in system settings
  • Whether biometrics are enabled in the app's settings
  • Whether facial/fingerprint data has been registered

If it still doesn't work, you can log in using your master password.

Where is the data stored?

All data is strictly stored locally on your device:

  • Encrypted data file: The app sandbox's Documents directory
  • Master password (if biometrics are enabled): The iOS system Keychain

Data will not be uploaded to the cloud or any server.

How do I change the master password?

  1. Open the app and go to "Settings"
  2. Tap "Change Master Password"
  3. Enter the current master password
  4. Enter the new password and confirm it

After changing, if biometric authentication is enabled, the password in the Keychain will be entirely automatically updated.

Is the app secure?

Yes, we employ multiple security measures:

  • AES-256 Encryption: Military-grade encryption standard
  • PBKDF2 Key Derivation: Prevents brute-force attacks
  • Fully Offline: Data never leaves the device
  • System Keychain: Biometric passwords are protected by the system
  • No Network Permissions: Incapable of connecting to the internet

Is iCloud synchronization supported?

No. To maximize security, this app runs entirely offline and does not support any cloud synchronization features.

If you need to sync across multiple devices, please use the "Export/Import" feature to manually transfer data.

Troubleshooting

App crashes or becomes unresponsive

  1. Force quit the app
  2. Restart your device
  3. Reopen the app

If the problem persists, please contact technical support.

Unable to save a new password

Please ensure:

  • The title and password fields are not empty
  • There is enough storage space on the device
  • The app has write permissions
